# 🗺️ みっけ！ | 協働学習プラットフォーム

**「誰もが迷わない直感UI × 先生の管理をゼロにする自動化 × 対話が生まれる協働空間」**

みっけ！は、Google Apps Script (GAS) と React で構築された、教育現場のための全く新しいマップ型協働学習プラットフォームです。サーバーの契約や複雑なデータベース構築は一切不要。コードをコピペするだけで、あなたのGoogleドライブ上にエンタープライズ級の教育アプリが全自動で誕生します。

## ✨ 革命的なコア機能

### 1. 🪄 ゼロコンフィグ ＆ 自動名簿ログイン

初回アクセス時に、裏側のスプレッドシート（データベース）や画像保存フォルダを全自動で構築。児童は自分のGoogleアカウント（メールアドレス）でアクセスするだけで、パスワード入力なしで一瞬で自分の名前にログインできます。

### 2. 📱 大企業SaaSレベルのモダンUI (SPA)

React と Tailwind CSS を活用し、Chromebook や iPad で絶対に崩れない、サクサク動く（60fps）美しいインターフェースを実現。全ての漢字には美しいベースラインで「ふりがな」が振られ、低学年の児童でも直感的に操作できます。

### 3. 📍 多次元マップ ＆ なぞって選択（投げ縄）

「町探検の予想マップ」と「結果マップ」など、同一単元内で複数の地図をタブで切り替え可能（ピンの引き継ぎもOK）。さらに、地図上をフリーハンドで囲むことで、そのエリアのピンだけを抽出し、共通点を見つけ出す高度な分析が可能です。

### 4. 💬 地図と連動する「みんなのひろば（チャット）」

ピンに対するコメントや、全体チャット、特定の発言への返信（リプライ）、そして「👍（いいね）」リアクション機能を搭載。学び合いが自然発生する協働空間を提供します。

### 5. 🤖 AI（Gemini）による自動ポートフォリオ生成

児童が刺したピンの内容やチャットでの発言をGoogleの最新AI（Gemini）が自動分析。「興味関心の傾向」や「先生への報告」を瞬時に生成し、ワンクリックで美しいレイアウトのままPDF印刷が可能です。先生の見取り・評価業務を劇的に効率化します。

## 🛠️ 技術スタック

*   **Frontend:** React 18, Babel (Standalone), Tailwind CSS
    
*   **Backend:** Google Apps Script (GAS)
    
*   **Database:** Google Spreadsheet (独自の疑似リレーショナルDB＆オートリカバリー機能付き)
    
*   **Storage:** Google Drive
    
*   **AI Integration:** Google Gemini API
    

※全て CDN と GAS だけで完結しているため、ビルド環境（Node.js等）は不要です。

## 🚀 導入方法（たったの3ステップ）

プログラミング未経験の先生でも、**5分**で導入完了します。

### Step 1: GASプロジェクトの作成とコードの貼り付け

1.  あなたの Google Drive を開き、「新規」＞「その他」＞「Google Apps Script」を作成します。
    
2.  プロジェクト名を「みっけ！Pro」に変更します。
    
3.  デフォルトの `コード.gs` を空にして、本リポジトリの `code.gs` の中身を貼り付けます。
    
4.  「＋」ボタンから「HTML」を選択し、ファイル名を `index` （`.html`は自動付与）にして作成。中身を本リポジトリの `index.html` に丸ごと入れ替えます。
    
5.  💾（保存）ボタンを押します。
    

### Step 2: デプロイ（アプリの公開）

1.  右上の青い「デプロイ」＞「新しいデプロイ」をクリックします。
    
2.  歯車マーク(⚙️)から「ウェブアプリ」を選択します。
    
3.  以下の通り設定し、「デプロイ」を押します。
    
    *   **実行するユーザー:** 「ウェブアプリにアクセスしているユーザー」
        
    *   **アクセスできるユーザー:** 「(あなたの学校のドメイン)のユーザー」 または 「Google アカウントを持つ全員」
        
4.  *※初回のみ「アクセスを承認」の画面が出ます。「詳細を表示」＞「安全ではないページへ移動」＞「許可」と進めてください。*
    

### Step 3: 初回アクセスと先生の設定

1.  デプロイ完了画面に出た **「ウェブアプリのURL」** をブラウザで開きます。
    
2.  開いた瞬間、あなたのGoogleドライブ内にデータベース用のスプレッドシートが自動作成され、あなたは「最初の先生」として登録されます。
    
3.  右上の「管理パネル」を開き、名簿（メアド,氏名,班）と単元の地図を登録すれば、すぐに授業で使えます！
    

## ⚙️ AI機能（Gemini API）の有効化

学習分析ダッシュボードを利用するには、無料のAPIキーが必要です。

1.  [Google AI Studio](https://aistudio.google.com/app/apikey "null") にアクセスし、「Create API key」から新しいキーを取得します。
    
2.  アプリの「管理パネル」＞「⚙️ AI設定」タブを開きます。
    
3.  取得したAPIキーを貼り付けて保存するだけで、AI機能が解放されます。
    

## 💡 授業での活用アイデア

*   **地域・町探検:** お気に入りスポット、危険な場所の共有マップ。
    
*   **修学旅行・校外学習:** 班ごとの見学記録、リアルタイムの現在地・発見共有。
    
*   **理科・生活科:** 校庭の植物や虫の生息地マッピング（スタンプピン活用）。
    
*   **防災教育:** 避難経路や消火器の場所を登録し、みんなで防災マップを作成。
    
*   **合意形成（主権者教育）:** 「ここに何を作るべきか？」をピンで提案し合い、チャットで議論。
